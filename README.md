# Lowbar

A re-implementation of many of the functions in the underscore library.

1. identity
2. first
3. last
4. each
5. indexOf
6. filter
7. reject
8. uniq
9. map
10. contains
11. pluck
12. reduce
13. every
14. some
15. extends
16. defaults

# Advanced Lowbar

1. `once`
2. `negate`
3. `shuffle`
4. `invoke`
5. `sortBy` (NB the Underscore library uses the native JavaScript sort but feel
   free to use your sort algorithm!)
6. `zip`
7. `sortedIndex`
8. `flatten`
9. `intersection`
10. `difference`
11. `memoize`
12. `delay`
13. `where`
14. `throttle`
15. `partial`
